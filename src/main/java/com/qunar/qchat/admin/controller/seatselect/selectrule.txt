没有机器人配置，没有真人客服服务记录的，其他伙伴可接客 －> 分配个给其他伙伴

在组内分配原则
一键分配阶段，如果指定产品，产品绑定服务组，只在组内分配客服
一键分配阶段，如果指定产品，产品未绑定服务组，在店铺内分配客服
＊一键分配阶段，未指定产品，只在店铺内分配客服，不考虑服务组的情况
＊分配客服的时候只分配 在线 或者 忙碌 的客服
